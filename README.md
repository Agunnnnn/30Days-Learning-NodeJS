# 🚀 30 Days Learning Node.js

## 📖 Deskripsi

Repository ini berisi proses pembelajaran Node.js selama 30 hari.
Setiap folder merupakan dokumentasi materi, eksperimen, dan latihan yang dilakukan secara bertahap dari dasar hingga pembuatan mini project.

---

## 🎯 Tujuan

* Memahami dasar-dasar Node.js
* Mempelajari penggunaan NPM dan package
* Mengembangkan kemampuan dalam mengelola data dan membuat program sederhana
* Meningkatkan konsistensi belajar dalam pemrograman
